# DBANK - Premium DeFi Banking Platform

A DeFi application with enterprise-level design, animated backgrounds, and real-time interest tracking.

## Features

- **Wallet Connection** - Connect wallet to start
- **Deposit/Withdraw** - Add or remove ETH funds
- **Real-time Interest** - Watch your balance grow (100ms updates)
- **Stats Dashboard** - 6 key metrics (Users, ETH, Rewards, Hold Days, Transactions, Security)
- **Transaction History** - Track all transactions
- **Premium UI** - Animated orbs, gradient effects, 4px borders, smooth animations

## Quick Start

```bash
git clone https://github.com/Vansh2309/DeFi.git
cd defi-dbank
open index.html
```

Or run with Python:
```bash
python -m http.server 8000
```

## Files

- `index.html` - Application markup
- `styles.css` - Styling & animations
- `script.js` - Core logic
- `package.json` - Project metadata

## Tech Stack

- HTML5, CSS3, Vanilla JavaScript
- No frameworks or dependencies
